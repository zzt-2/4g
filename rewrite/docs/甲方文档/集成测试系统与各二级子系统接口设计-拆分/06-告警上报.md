### 设备告警上报

**使用场景：**

二级子系统设备告警上报。

**流程示例：**

![](media/image18.emf)

图 4‑22设备告警上报流程

接口说明

  -----------------------------------------------------------------------
  url              https://127.0.0.1:5001/api/
                   deviceInfo/deviceAlarmReport
  ---------------- ------------------------------------------------------
  协议             https

  请求方式         POST
  -----------------------------------------------------------------------

> **接口结构：**

请求消息参数表

  ------------------------------------------------------------------------------------------------------------------------------------------------------
  **英文名称**         **数据类型**   **要求**   **备注**
  -------------------- -------------- ---------- -------------------------------------------------------------------------------------------------------
  method               字符串         必选       接口名称：deviceAlarmReport

  requestId            数字           必选       请求id，取值范围：0\~2147483648。说明：用于关联请求和响应消息，由请求方生成，响应方法中必需回填此值。

  subSysType           字符串         必选       二级子系统类型，类型：字符串，取值范围：子系统标识符，见系统分类表。。

  subSysId             字符串         必选       二级子系统id，类型：字符串，取值范围：小于15个字符。说明：子系统的唯一标识id。

  sessionId            数字           必选       类型：int，取值范围：0\~2147483648。说明：流程上下文id

  datas                数组           必选       

  data                 对象                      

  \--data\--alarmId    字符串         必选       告警编号，字符串，取值范围：小于10个字符

  \--data\--deviceId   字符串         必选       设备id，唯一标识一个设备，类型：字符串，取值范围：不大于50个字符。建议格式：
                                                 子系统标识_设备类型_编号,如：ADS_UE_01。

  \--data\--severity   字符串         必选       告警级别，字符串，取值范围：\"critical:严重告警;major:主要告警;warn:警告告警;clear:清除告警\"。

  \--data\-- warnTime  字符串         必选       告警产生时间

  \--data\--msg        字符串         必选       携带告警信息,类型：字符串，取值范围：小于等于150字符。
  ------------------------------------------------------------------------------------------------------------------------------------------------------

响应消息参数表

  -------------------------------------------------------------------------------------------------------------------------------------------------
  **英文名称**    **数据类型**   **要求**   **备注**
  --------------- -------------- ---------- -------------------------------------------------------------------------------------------------------
  method          字符串         必选       接口名称：deviceAlarmReportResponse

  requestId       数字           必选       请求id，取值范围：0\~2147483648。说明：用于关联请求和响应消息，由请求方生成，响应方法中必需回填此值。

  subSysType      字符串         必选       二级子系统类型，类型：字符串，取值范围：子系统标识符，见系统分类表。。

  subSysId        字符串         必选       二级子系统id，类型：字符串，取值范围：小于15个字符。说明：子系统的唯一标识id。

  sessionId       数字           必选       回填请求消息中的sessionId

  statusCode      整形           必选       响应状态码，类型：int,取值范围：\"1:正常;2:异常\"。

  msg             字符串         必选       正常或异常信息,类型：字符串，取值范围：小于等于150字符。
  -------------------------------------------------------------------------------------------------------------------------------------------------
